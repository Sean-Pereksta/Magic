How this solo endless mode works

Core loop
  1. Each round has a target score. Beat it to survive.
  2. Rounds cycle Small → Big → Boss (Boss is every 3rd round).
  3. Boss rounds add limitations (no discards, exact 5 cards, suit demands, etc.).
  4. Once you reach target, you may keep scoring or hit Finish Round to shop.
  5. No timers: the run ends only if you run out of hands before reaching the target (or you finish below target).
Controls
Tap cards to select. Press Esc to close modals / drawer (desktop).

Difficulty
